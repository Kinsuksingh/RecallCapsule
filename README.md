# RecallCapsule

**RecallCapsule** is an AI-powered study tool designed to help you master any subject through a tailored, adaptive curriculum.

## Key Features

- **Capsules**: Bite-sized, searchable learning modules for self-paced study. You can browse and filter these AI-crafted educational modules by topic and grade level to master specific subjects.
- **Calls**: Connect with specialized AI tutors for real-time, interactive voice guidance. You can view available tutors, their subject expertise, and initiate a hands-on, personalized tutoring session instantly.
- **Labs**: Virtual interactive environments where you can safely conduct detailed scientific experiments and simulations without limits.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
